# backend-dev2

Building a backend API for a bootcamp project from a tutorial.

## Usage
Rename "config/config.env.env" to "config/config.env" and update the value and settings to your own

## Install Dependcies
```
npm install
```

## Run App
```
#Run in dev mode
npm run dev

#Run in prod mode 
npm start
```
